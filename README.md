# Checkbox input

* A custom checkbox input which uses an inline SVG for the checked state.
* The SVG scales along when the checkbox is resized.
* The attibutes `role="presentation"` and `aria-hidden="true"` have been added to the SVG to prevent old screen readers from reading the SVG source code.

## Demo

View the [demo](https://voorhoede-checkbox-input.netlify.com/).

## Keyboard support

* Tab - Moves keyboard focus to the checkbox.
* Space - Toggles checkbox between checked and unchecked states.
